# Business-Performance-Dashboard
This project demonstrates my hands-on skills in data analysis using Microsoft Excel. The dataset was cleaned, analyzed, and transformed into meaningful insights using Pivot Tables, followed by the creation of an interactive dashboard.

### Key Features:
- Data cleaning and structuring in Excel

- Pivot Tables for dynamic analysis

- KPI-based insights and summaries

- Interactive dashboard for quick decision-making

- MIS-style reporting format

### Tools Used:
- Microsoft Excel

- Pivot Tables

- Charts & Visualizations
